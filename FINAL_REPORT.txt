Quick Start
-----------

Start the daemon:
  $ ./start_daemon.sh start

Check status:
  $ ./start_daemon.sh status
  $ python3 health_check.py --status

View logs:
  $ ./start_daemon.sh logs

Stop the daemon:
  $ ./start_daemon.sh stop

Setup auto-monitoring (cron):
  $ crontab -e
  Add: */2 * * * * cd /home/openclaw/FinRobot && /usr/bin/python3 health_check.py

New Log Format
--------------

[ITER: 382/10000 3.8%] Price: 4566.70 | G:-0.45%|19.9%|5 M:-0.99%|19.9%|7629 H:0.00%|0.0%|0 | Best:M

This shows:
- Progress: Iteration 382 of 10,000 (3.8% complete)
- Price: 4566.70
- Grid (G): -0.45% return, 19.9% win rate, 5 trades
- Martingale (M): -0.99% return, 19.9% win rate, 7629 trades
- HFT (H): 0.00% return, 0.0% win rate, 0 trades
- Best: M (Martingale is currently best)
